# Roboconf

Like `autoconf`, but for Ruby.

## Installation

Add this line to your application's Gemfile:

    gem 'roboconf', '~> 0.0.1', :git => 'git@github.com:hedgeyedev/roboconf.git'

And then execute:

    $ bundle install --binstubs

Or install it yourself as:

    $ gem install roboconf
